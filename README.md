# CD Transports Website

This is the production build for **cdtransports.com**.

## Contents
- Fully responsive static site
- Hero video background (desktop)
- Mobile-optimised layout
- SEO meta tags
- Open Graph image for social sharing
- Favicons for Google Search
- Sitemap for indexing
- robots.txt file

## Deployment
Upload the contents of `cd-transports-hero-video-redesign/` to your hosting root.

Place `sitemap.xml` and `robots.txt` in the root as well (same level as index.html).
